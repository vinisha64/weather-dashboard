# 🌦️ Weather Dashboard

A responsive weather dashboard that provides real-time weather information for any city using the OpenWeatherMap API. The application features dynamic weather animations, modern glassmorphism UI, and live weather metrics.

## 🌐 Live Demo

🔗 https://vinisha64.github.io/weather-dashboard/

## ✨ Features

- Search weather by city name
- Real-time weather information
- Current temperature and "Feels Like" temperature
- Humidity, wind speed, wind direction, and pressure
- Dynamic weather icons
- Animated weather backgrounds based on conditions
- Responsive design for desktop and mobile devices
- Error handling for invalid city names

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/vinisha64/weather-dashboard.git
```

### Navigate to the project

```bash
cd weather-dashboard
```

### Run the project

Open `index.html` in your preferred web browser.
